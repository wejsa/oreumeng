# 오름이엔지 홈페이지

시스템실링, 크린부스, FFU 설치 전문 시공업체 **오름이엔지** 공식 홈페이지

## 프로젝트 개요

- **회사명**: 오름이엔지
- **대표**: 심지은
- **설립일**: 2018년 1월 1일
- **주요 서비스**: 시스템실링, 크린부스, 판넬타입, 커튼설치, 파티션
- **웹사이트**: https://wejsa.github.io/oruemeng/

## 기술 스택

- **HTML5** - 시맨틱 마크업
- **CSS3** - CSS Variables, Flexbox, Grid, 반응형 미디어 쿼리, 애니메이션
- **JavaScript (Vanilla)** - 모바일 메뉴, 포트폴리오 필터, 라이트박스
- **외부 리소스**
  - Google Fonts (Noto Sans KR)
  - Kakao Maps Static API

## 프로젝트 구조

```
    ├── Partition/          # 파티션 시공사례 (3장)

```
